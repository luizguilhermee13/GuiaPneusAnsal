Guia Ansal - Gestão de Pneus

Sistema desenvolvido para centralizar e disponibilizar guias operacionais relacionados à gestão de pneus da Ansal JF, facilitando o acesso rápido a procedimentos, treinamentos e materiais de apoio para colaboradores da área.

O projeto foi criado com foco em organização, acessibilidade e continuidade operacional, permitindo que novos funcionários tenham acesso aos processos e instruções de forma prática e visual.



Tecnologias Utilizadas
HTML5
CSS3
JavaScript

Funcionalidades
Landing page responsiva
Menu de navegação fixo
Seção de apresentação do projeto
Cards dinâmicos para exibição dos guias
Formulário de sugestões integrado ao WhatsApp
Layout adaptado para desktop, tablet e mobile
Redirecionamento para materiais em PowerPoint
