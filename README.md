# 🛞 Guia Ansal - Gestão de Pneus

Sistema desenvolvido para centralizar e disponibilizar guias operacionais relacionados à gestão de pneus da **Ansal JF**. O foco é facilitar o acesso rápido a procedimentos, treinamentos e materiais de apoio para colaboradores da área.

> [!IMPORTANT]
> O projeto foi criado com foco em organização, acessibilidade e continuidade operacional, permitindo que novos funcionários tenham acesso aos processos e instruções de forma prática e visual.

---

## 🚀 Tecnologias Utilizadas

O projeto foi construído utilizando as seguintes tecnologias:

*   **HTML5**: Estruturação de conteúdo.
*   **CSS3**: Estilização e layout responsivo.
*   **JavaScript**: Lógica de interatividade e integração.

---

## ✨ Funcionalidades

- ✅ **Landing page responsiva**: Navegação fluida em qualquer dispositivo.
- 📌 **Menu de navegação fixo**: Acesso rápido às seções a qualquer momento.
- 🖼️ **Cards dinâmicos**: Exibição organizada e visual dos guias.
- 💬 **Integração com WhatsApp**: Formulário de sugestões direto para o suporte.
- 📱 **Layout Adaptável**: Totalmente otimizado para Desktop, Tablet e Mobile.
- 📊 **PowerPoint Ready**: Redirecionamento direto para materiais de treinamento.
